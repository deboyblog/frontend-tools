## 基于ElectronVue + VueBlu

只为身为前端狗的我打造一套专属的前端工具集合桌面端

| 进度 | 功能 | 描述 |
|---|---|---------|
| [x] | Px2rem | 可配置UI设计图像素(待转换像素) + 目标基准像素 如 750px宽度的设计图 640px宽度的REM方案 转换方式为 目标值(rem) =  待转换像素 * (640 / 750)  


目前只针对我自身需求来定制
