# 前端工具集合
## 基于ElectronVue + VueBlu

只为打造一套专属的前端工具集合桌面端

| 进度 | 功能 | 描述 |
|---|---|---------|
| [√] | Px2rem | 750px的UI图 640px的REM方案 1rem=100px的转换率 公式 **目标值(rem) =  待转换像素 * (640 / 750) / 转换率**
| [√] | 自定义布局 | 通过传递x,y,showSlider来控制app的布局转换

目前只针对我自身需求来定制
